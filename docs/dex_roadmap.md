## 고려 중인 DEX

### Hyperliquid (최우선 후보)
- 메이커 수수료: -0.002% (리베이트, 수익)
- 테이커 수수료: 0.05%
- 가스비: 없음
- 속도: ~500ms (온체인 오더북)

## 통합 전략

### CEX 현물 + Hyperliquid 선물 펀딩피 아비트라지
```
Binance/Bybit 현물 매수 (테이커 0.1%)
+ Hyperliquid 선물 숏 (메이커 주문 시 -0.002% 리베이트)
= 수수료 합계: 0.098% (기존 0.15% 대비 절감)
```

### 예상 수익 개선
- 기존 (CEX+CEX): 펀딩피 - 0.15% (왕복)
- DEX 포함 시:  펀딩피 - 0.098% (왕복)
- 개선폭: ~35% 수수료 절감
